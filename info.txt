--- Как запустить терминал

Ctrl + Alt + T

--- Как выключить компьютер

sudo shutdown -h now

--- Как перезагрузить компьютер 

sudo reboot -h now

--- Вставить скопированный текст в терминал

Ctrl + Shift + v вместо Ctrl + v

--- Скопировать из терминала 

Ctrl + Shift + c вместо Ctrl + c

--- Как развернуть терминал в полный экран

Alt + F10

--- Как поменять язык (русский или английский)

Windows + пробел

--- Как открыть новую вкладку в терминале

Ctrl + Shift + T

--- Как закрыть вкладку в терминале

Ctrl + d

--- Как переключаться между вкладками в терминале

Alt + 1, Alt + 2, ... -- быстрое переключение на первую/вторую/третью вкладку

Ctrl + PageUp/Ctrl + PageDown -- переход на следующую/предыдудую вкладку (надо использовать кнопки PageUp/PageDown которые находятся в верхнем правом углу клавиатуры)

--- Как открыть текстовый редактор

vim file.py -- создает новый файл file.py или открывает уже созданный. Название файла можно выбирать любым.

--- Как работать с vim

1) esc -- переход в режим команд

   - команда :w -- сохранить изменения
   - команда :q -- выйти
   - команда :q! -- выйти без сохранения файла
   - команда :wq -- сохранить и выйти
   - ctrl + a -- запустить программу (до запуска надо всегда использовать :w чтобы сохранить программу, иначе запустится старая несохраненная версия)

2) i -- переход в режим редактирования

--- Как пользоваться git

1) git add file.py  добавить файл

2) git diff --cached показывает изменения перед коммитом

3) git commit закоммитить (зафиксировать проделанную работу) 

4) git push отправить все коммиты на github, чтобы другие люди могли смотреть изменения
